# Insurance Premium Prediction

This project predicts insurance premiums using Machine Learning.

## Files
- app.py - Main application
- model.pkl - Trained Machine Learning model
- scaler.pkl - Data scaler
- insurance[1].csv - Dataset

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
